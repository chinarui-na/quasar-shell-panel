# About

#### 此项目是使用[quasar](https://quasar.dev/)，是桌面级的ssh和sftp客户端。
#### 支持ssh/sftp，文件上传、下载、重命名、删除。文件（夹）拖拽上传；上传、下载进度查看；在线编辑文件；
